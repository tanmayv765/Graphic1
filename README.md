# Stock Return Forecaster

A lightweight, dependency-free website that projects how an investment could grow over time for a selected stock profile.

## Features

- Stock presets for Apple, Microsoft, NVIDIA, Amazon, and the S&P 500 ETF
- Adjustable expected annual return
- Inflation input to show inflation-adjusted values in today's dollars
- Initial investment and monthly contribution inputs
- Live metrics for ending value, total invested, gain, and return multiple
- SVG growth chart and yearly projection table
